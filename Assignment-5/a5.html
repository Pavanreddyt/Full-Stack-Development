What is Web Storage?

localStorage and sessionStorage are part of the Web Storage API in HTML5. They allow websites to store data in a user's browser.

localStorage stores data with no expiration time — it persists even after the browser is closed. This is useful for saving user preferences, themes, or cached data.

sessionStorage stores data for the duration of the page session — the data is cleared when the page is closed. This is often used for temporary data like form progress or tab-specific information.
